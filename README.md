# introduction-to-data-science-final-project

En este repositorio llevamos la documentacion y codigo de nuestro proyecto exploratorio de datos.

Que datasets investigamos:

https://www.kaggle.com/datasets/shaistashahid/data-science-jobs

https://www.kaggle.com/datasets/hussnainmamoon1/llm-hallucination-benchmark-v2-25k-rows-6-models

https://www.kaggle.com/datasets/karkavelrajaj/amazon-sales-dataset

Que incluye este proyecto:

-La limpieza de un dataset completo.

-La exploracion de este mismo con analisis grafico.

-Conclusiones en base a correlacion de variables y otros factores.

-Modelo de regresion lineal predictivo para el dataset en cuestion.
